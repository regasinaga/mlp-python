# Multi Layer Perceptron Implementation

Example of multi layer perceptron (neural network) implementation in python. This code can be used for 
semi-deep learning.

## Getting Started

### Prerequisites

These libraries must be installed

* Numpy
* SciPy
* Matplotlib

## Built With

* Python 3.4
